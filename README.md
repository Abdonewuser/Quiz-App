# Quiz Application — React

An interactive Quiz Application built using React that allows users to test their knowledge through a series of multiple-choice questions with real-time scoring. The application demonstrates dynamic state updates, conditional rendering, and single-page navigation using smooth scrolling.

---

## 🚀 Features

- Multiple-choice quiz questions  
- Real-time score tracking  
- Dynamic question rendering  
- Conditional rendering for quiz flow  
- Smooth section navigation using `scrollIntoView()`  
- Single-page application behaviour  

---

## 🛠 Tech Stack

- React (Hooks)  
- JavaScript (ES6+)  
- HTML5  
- CSS3  

---

## 🎯 What I Learned

- Managing quiz state and score tracking in React  
- Implementing smooth scrolling navigation using `scrollIntoView()`  
- Creating single-page behaviour without traditional routing  
- Understanding navigation state concepts in `NavLink` and `useLocation`  
- Exploring scroll-based animation techniques using `IntersectionObserver`
